## Sudoku Tkinter アプリ

このリポジトリは、Python と Tkinter で作られた数独（Sudoku）デスクトップアプリです。

## 主な機能

- 9x9 の数独盤面表示
- 難易度選択（Easy / Normal / Hard）
- 新規ゲーム、リセット、チェック、ヒント
- メモ（候補数字）入力モード
- タイマー、ミス数、入力済みマス数の表示
- セーブ / ロード（JSON）

## セットアップ

### 前提

- Python 3.10 以上（推奨）
- Tkinter が利用可能な環境

### 実行方法

リポジトリルートで以下を実行します。

```bash
python -m src.sudoku_app.app
```

## テスト

```bash
python -m unittest discover -s test -p '*.py'
```
